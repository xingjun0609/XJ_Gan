Smart Japanese Heater Control System

Smart IoT devices have gained popularity, with examples such as air conditioners from Panasonic and Daikin. However, traditional Japanese heating devices, like the 'Kotatsu,' still rely on wired controls. Recognizing this technological gap, the Smart Japanese Control System emerges as a pioneering solution. This cross-platform Flutter and Dart application, supported by Firebase, not only bridges the divide but also showcases my proficiency in modern app development. NodeJS seamlessly integrates 'Firebase Cloud Messages' for efficient notification delivery. Real-time control and monitoring ensure dynamic updates, fostering user convenience. A responsive UI guarantees optimal performance across devices. Moreover, the integration of ESP32 as an IoT control device reflects my commitment to delivering advanced, user-centric solutions through innovative features and cutting-edge technologies.

Vaccination Registration Platform

The Vaccination Registration Platform is an Android mobile app developed with Java and Android Studio. It features client and admin modules for registration and data management, allowing users to easily register themselves and their dependents for vaccination. This project expanded my skills in mobile app development and created an efficient platform for vaccination registration.

Agricultural Lighting Schedule and Control System

The Agricultural Lighting Schedule and Control System, integrating Arduino and HTML JavaScript technologies, excels as a tailored and user-friendly tool for low-quantity client networks. Designed for Singaporean indoor plant enthusiasts, it prioritizes easy installation, intuitive operation, and accessibility. This versatile system offers real-time control over light brightness, scheduling, and efficient water pump control for precise agricultural irrigation. Utilizing ESP32 for output control, it incorporates SPIFFF and EPPROM storage with AP (Wireless Control), enabling wireless local area network control without costly installations. Its adaptability extends to laboratory light control, showcasing versatility in agricultural and scientific settings. The fast implementation solution underscores its commitment to streamlined management in indoor plant cultivation and beyond.

Emergency Light Monitoring System

The IoT-based Emergency Light Monitoring System, developed on Arduino ESP32 for Android mobile devices, revolutionizes emergency lighting management. Beyond basic controls and battery monitoring, the system introduces Weekly Self Testing, ensuring ongoing reliability by conducting regular self-checks. Group Control facilitates centralized management of multiple lights, enhancing efficiency in larger spaces. Additionally, the system's Report Generation feature automates the creation of detailed reports, offering valuable insights for maintenance and performance optimization. This comprehensive solution not only deepened my expertise in IoT technology but also represents a significant advancement in creating a reliable, efficient, and user-centric emergency lighting system.
